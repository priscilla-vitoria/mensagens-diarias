# Mensagens Diarias
Programa em Java que exibe uma mensagem motivacional baseada no dia da semana escolhido pelo usuario (1 a 7).
## Como rodar
 1. Tenha o Java instalado no seu computador.
 2. Baixe o arquivo MensagensDiarias.java.
 3. Importe o arquivo em sua IDE de preferência e execute a classe main.
